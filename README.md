# Monmon's Birthday Adventure 🎂

A Nintendo-inspired birthday adventure game made with HTML, CSS, and JavaScript.

## Features
- Pixel-art style
- Mini games
- Music and sound effects
- Multiple scenes
- Final boss battle

## How to Play
1. Open the website.
2. Complete each mission.
3. Defeat the final boss.
4. Enjoy the ending!

## Technologies Used
- HTML5
- CSS3
- JavaScript
